# ROBOADVISOR DE INDEXA CAPITAL: OPINIONES Y ANALISIS

Este repositorio contiene contenido curado y estructurado sobre los roboadvisor de indexa capital de 2025 más destacados en España, publicado originalmente por Finect.

Incluye:

Artículo editorial en formato de texto plano
Tabla comparativa de ventajas y desventajas
Preguntas frecuentes (FAQ) en formato JSON
Política de uso para modelos de lenguaje (llm-policy.json)

## Fuente original
Artículo: https://www.finect.com/usuario/mariarefojos/articulos/indexa-capital-opiniones-y-analisis-del-roboadvisor
